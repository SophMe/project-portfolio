    • JavaScript Basics

    JavaScript uses variables, data types, and type checking for basic operations.

    Variables can be declared using var, let, and const.

    var x = 10;
    let y = 20;
    const z = 30;
  • Primitive data types: String, Number, Boolean, Null, Undefined.
  • Reference types: Object, Array, Function.
    • Functions and Scope

    Functions can be declared in multiple ways, including function declarations, expressions, and arrow functions.

    JavaScript also supports different types of scopes: global, function, and block scope.

    function greet() { return "Hello!"; }
    const greet = () => "Hello!";
  • Variables declared with var are function-scoped.
  • let and const are block-scoped.
    • Objects and Arrays

    Objects are collections of key-value pairs, while arrays are ordered lists of values.

    Objects and arrays are essential for working with structured data.

    const person = { name: "John", age: 30 };
    const numbers = [1, 2, 3, 4];
  • Access object properties with dot notation or square brackets.
  • Use array methods like push(), pop(), shift(), and unshift().
    • Control Flow

    JavaScript supports conditional statements like if and switch, and loops for repeated execution.

    Common loops include for and while.

    if (age >= 18) { console.log("Adult"); }
    for (let i = 0; i < 5; i++) { console.log(i); }
  • switch statements allow multiple conditions to be checked.
  • while loops execute as long as a condition is true.
    • Advanced Topics

    Modern JavaScript (ES6+) introduced powerful features like destructuring, the spread operator, and promises for async code.

    Async/await provides a cleaner syntax for handling asynchronous operations.

    const [a, b] = [1, 2];
    let promise = new Promise((resolve, reject) => resolve("Success!"));
  • Destructuring allows extracting data from arrays and objects.
  • Promises and async/await are essential for handling asynchronous tasks.
